Surgimento da Internet

Criada pelo britânico TBL o acrônimo HTML significa Hyper Text Markup Language (em português, Linguagem de Marcação de Texto). O HTML é componente básico da Web, permitindo inserir o conteúdo e estabelecer a estrutura básica de um website. Sem ele, o navegador não saberia como exibir textos ou outros conteúdos como imagens e vídeos por exemplo.

Os hipertextos de um website são conjuntos de elementos conectados, que podem ser palavras, imagens, vídeos, documentos, vídeos etc. Uma vez conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações.

Ao visitar uma página simples na web, pode-se notar que existem diferentes distribuições de tamanho para títulos, parágrafos, imagens, vídeos e qualquer outro tipo de elemento. Toda essa estrutura é estabelecida através do HTML. No início da web era comum encontrar sites apenas contendo texto e poucas imagens simples, sem estilizações, porém nos dias de hoje, é muito dificil encontrar um site contendo apenas elementos HTML. Considera-se então, o HTML como o "esqueleto" de suas páginas web.

Além do "esqueleto", também é necessário ter o "corpo". Para isso, temos as linguagens CSS e o JavaScript, que em junto do HTML formam a base de todos os websites atuais.

Como funciona o HTML

Através de um documento HTML (.html ou .htm), o navegador faz a leitura do arquivo e renderiza para o usuário final. Este arquivo pode ser visualizado em qualquer navegador (Chrome, Edge, Opera, etc.)

Um site, normalmente, é composto por diversas páginas HTML, exemplo: um site que contenha 4 páginas (uma landing page, uma página sobre, uma página de contatos e outra de produtos). Este site, por exemplo, receberá 4 documentos .html distintos para cada página, respectivamente.

O código pode ser escrito em qualquer editor de texto, como o próprio TextPad. Cada página de um documento .html é composta por uma série de elementos (tags), considerados "blocos" de construção das páginas. Este "blocos", é a maneira a qual o HTML faz a marcação dos conteúdos, criando a hierarquia e a estruturação dos mesmos, dividindo entre seções, parágrafos, cabeçalhos e muito mais

O que são tags

Através de qualquer editor de textos, é possível criar um documento com a extensão .html que será renderizado por qualquer navegador.

Este documento consiste em uma série de tags. As tags são códigos que definem toda a estrutura da página, como por exemplo: tamanho, fonte da letra, quebras de linhas, etc. A maioria dos elementos de um documento HTML são compostas por uma estrutura de abertura e de fechamento, como a tag p, h1, a, etc. Existem também tags de estrutura única como a tag hr/ que cria uma linha divisória de conteúdo e a tag br, que realizar a quebra da linha.

Por exemplo, ao criar um título com a tag h1, chamamos a mesma escrevemos nosso título e encerramos com /h1. Desta maneira, ao salvar e executar o arquivo .html criado, será possível visualizar o título criado sendo exibido no navegador.
